### Boruvka's Algorithm

Vertex u:     

    p (pointer to the other endpoint of the shortest edge out of u) 
    Initially P(u) = u
    al (Adjacency list)

Edge (u, v):  

    Next, l, v1, v2


    edge pardo:
      tnext = next;
      min = l;

      repeat:
        min = min[tnext.min, min];
        tnext = tnext.tnext;
      until:
        tnext = nil; done = true;

Sometimes, all processors may be done early!

    vertex pardo:
      min = al.min;
    edge pardo:
      if v1.min = l and v1.p = v2, tree = true
